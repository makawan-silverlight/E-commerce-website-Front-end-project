# SILVER ECOMMERCE STORE WEBSITE
Frontend Project Ecommerce website

create website using:
- React TS (vite)
- Tailwind CSS
- Type Script
- React-Hook Form
- React-Router-Dom
- Axios
- Redux-Toolkit
- Swiper
- React-Icons

### API
- https://dummyjson.com/products
